How to HTML

HTML, which stands for "HyperText Markup Language," is a markup language used in the construction and structuring of web pages. Developed to describe the content of documents, such as texts, images, and links, HTML uses tags to define elements and attributes that determine the presentation and functionality of these elements in a web browser.

HTML pages are the fundamental basis of the web, interpreted by browsers to display content visually comprehensible to users. Each element in HTML serves a specific function, such as paragraphs, headers, images, links, forms, among others. The combination and nesting of these elements allow for the creation of complex and interactive web pages. Additionally, HTML is often used in conjunction with other web technologies, such as CSS (Cascading Style Sheets) for styling and layout, and JavaScript to add interactivity and dynamism to pages. Together, these three components form the foundation for building and designing a significant portion of websites on the internet.

Basic Tags

Tags iniciais

<!DOCTYPE html>
The <!DOCTYPE html> declaration is an instruction in HTML that defines the version of HTML used in the document. In this case, "html" refers to the latest version of HTML5. This statement helps browsers interpret the HTML code of the page correctly, ensuring proper rendering. In summary, <!DOCTYPE html> is an initial indication to browsers about the HTML version being used in the document.

<head>
The <head> tag in HTML is used to contain metadata and information about the document, which is not displayed directly on the page. It includes elements like <title>, which defines the page title displayed in the browser's title bar, and can also contain links to CSS style sheets, scripts, meta tags, and other elements essential for the formatting and behavior of the web page. In summary, the <head> tag is intended for important information about the page that is not visible to users but is crucial for the correct display and functioning of the content.

<body>
The <body> tag in HTML is used to contain the main content of a web page. All content visible to users, such as text, images, links, and other elements, is placed within this tag. In summary, <body> defines the body of the HTML document, where the content is presented and displayed on the web page.

Tables

The <table> tag in HTML is used to create tables. It encompasses tabular content and is composed of other tags, such as <tr> for table rows, <th> for table headers (optional), and <td> for data cells. In summary, <table> is used to structure and organize data in tabular format on a web page.

Images

The <img> tag in HTML is used to embed images and gifs on a web page. It has a mandatory attribute called src, which specifies the path to the image file. Additionally, other optional attributes, such as alt for alternative text and width/height to specify dimensions, can be included. In summary, <img> allows the display of images on a web page, playing an essential role in visual content.

Videos

The <video> tag in HTML is used to embed videos on a web page. It requires the src attribute to indicate the path to the video file. Other optional attributes, such as controls to display playback controls, width/height to set dimensions, and autoplay to automatically start playback, can be included. In summary, <video> enables the inclusion of video content on a web page, providing a multimedia experience for users.
