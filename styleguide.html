Vanilla Style

That's sounds really good huh?!

It's essential, you know, to have a style on your page. A nice color palette, some good fonts for you to use on different occasions. Just in case, you know, one doesn't work. And of course, not less important, styling each element on the page!

Well, as mentioned earlier, a good styling is needed to spruce up the look. So, I'm gonna teach the vanilla style by showing my own styling.

Stylishing

Now that we've covered the absolute basics, let me throw in some styling ideas that I'm actually rocking on this very site. Each of these simple tricks can be jazzed up in various ways for your own project.

Cards

Guys, as you can see, I really enjoy making cards to organize things, and, of course, it's a cool way to jazz up your site's vibe. So, I'm gonna show you how I do that.


HTML

<div class="card">
  <h5>HTML</h5>
  <p> Any text you want </p>
</div>

CSS

.card {
background-color: rgb(172, 131, 41, 0.5);
padding: 20px;
margin: 20px;
margin-top: 20px;
}

Nav Bar

The navigation bar is like the playground of website styling. It kinda sets the vibe for your site and always grabs attention. Whether it's a sleek and modern Navbar or something wild and vintage like the madness I've got going on, it's all fair game.


HTML

<nav>
  <ul>
    <li><a href="index.html">Home </a></li>
    <li><a href="guides.html">Guides </a></li>
  </ul>
</nav>

CSS

nav{
font-family: 'Signika';
font-size: 18px;
position: sticky;
top: 0;
z-index: 5;
}

nav ul{
list-style-type: none;
overflow: hidden;
background-color:rgb(172, 131, 41);
background-image: url(images/casquinha.png);
background-position: right;
background-size: contain;
background-repeat: no-repeat;
}

Animation : Chatioc Version (BE CAREFUL)

Here, I'm gonna show you how to spice things up with animations. This one's gonna be simple but delightfully chaotic. Just keep your mouse on it for the magic to happen!

HTML

<div>
  <span class="circular" id="example">
  </span>
</div>

CSS

.circular img{
text-align: center;
margin-left: 150px;
border-radius: 50%;
height: 300px;
box-shadow: 0px 2px 6px 0px white;
}

#example:hover{
animation: rotation 2s infinite linear;
}

@keyframes rotation {
from {
  transform: rotate(0deg);
  }
to {
  transform: rotate(359deg);
  }
}
