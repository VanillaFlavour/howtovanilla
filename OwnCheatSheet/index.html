How to HTML

HTML, que significa "HyperText Markup Language" (Linguagem de Marcação de Hipertexto), é uma linguagem de marcação utilizada na construção e estruturação de páginas web. Desenvolvido para descrever o conteúdo de documentos, como textos, imagens e links, HTML utiliza tags (etiquetas) para definir elementos e atributos que determinam a apresentação e o funcionamento desses elementos em um navegador web.

As páginas HTML são a base fundamental da web, sendo interpretadas pelos navegadores para exibir o conteúdo de forma visualmente compreensível para os usuários. Cada elemento em HTML possui uma função específica, como parágrafos, cabeçalhos, imagens, links, formulários, entre outros. A combinação e aninhamento desses elementos permitem a criação de páginas web complexas e interativas.

Além disso, HTML muitas vezes é utilizado em conjunto com outras tecnologias web, como CSS (Cascading Style Sheets) para estilização e layout, e JavaScript para adicionar interatividade e dinamismo às páginas. Juntos, esses três componentes formam a base da construção e design de grande parte dos sites na internet.

Basic Tags

Tags iniciais

< !DOCTYPE html >
A declaração < !DOCTYPE html > é uma instrução no HTML que define a versão do HTML utilizada no documento. No caso, "html" refere-se à versão mais recente do HTML5. Essa declaração ajuda os navegadores a interpretar corretamente o código HTML da página, garantindo a renderização apropriada. Em resumo, < !DOCTYPE html > é uma indicação inicial para os navegadores sobre a versão do HTML sendo usada no documento.

< head >
A tag < head > em HTML é usada para conter metadados e informações sobre o documento, que não são exibidos diretamente na página. Ela inclui elementos como < title >, que define o título da página exibido na barra de título do navegador, e pode conter também links para folhas de estilo CSS, scripts, metatags e outros elementos essenciais para a formatação e comportamento da página web. Em resumo, a tag < head > é destinada a informações importantes sobre a página que não são visíveis para os usuários, mas são cruciais para a correta exibição e funcionamento do conteúdo.

< body >
A tag < body > em HTML é usada para conter o conteúdo principal de uma página web. Todo o conteúdo visível para os usuários, como textos, imagens, links e outros elementos, é colocado dentro dessa tag. Em resumo, < body > define o corpo do documento HTML, onde o conteúdo é apresentado e exibido na página web.

Tables

A tag < table > em HTML é usada para criar tabelas. Ela engloba o conteúdo tabular e é composta por outras tags, como < tr > para linhas de tabela, < th > para cabeçalhos de tabela (opcional), e < td > para células de dados. Em resumo, < table > é utilizada para estruturar e organizar dados em formato tabular numa página web.

Medias

Images

A tag < img > em HTML é usada para incorporar imagens e gifs em uma página web. Ela possui um atributo obrigatório chamado src, que especifica o caminho do arquivo da imagem. Além disso, outros atributos opcionais, como alt para texto alternativo e width/height para especificar as dimensões, podem ser incluídos. Em resumo, < img > permite a exibição de imagens na página web, sendo uma parte essencial para o conteúdo visual.

Videos

A tag < video > em HTML é usada para incorporar vídeos em uma página web. Ela requer o atributo src para indicar o caminho do arquivo de vídeo. Outros atributos opcionais, como controls para exibir controles de reprodução, width/height para definir as dimensões, e autoplay para iniciar automaticamente a reprodução, podem ser incluídos. Em resumo, < video > possibilita a inclusão de conteúdo de vídeo em uma página web, oferecendo uma experiência multimídia aos usuários.
